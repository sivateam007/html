HTML Plug-ins

சேர்க்கை நிரல்கள் (Plug-ins) என்பது உலாவியின் நிலையான செயல்பாட்டை விரிவுபடுத்தும் கணினி நிரல்கள் ஆகும்.

சேர்க்கை நிரல்கள்

சேர்க்கை நிரல்கள் பல்வேறு நோக்கங்களுக்காகப் பயன்படுத்த வடிவமைக்கப்பட்டன:

  • Java பயன்பாடுகளை இயக்க
  • Microsoft ActiveX கட்டுப்பாடுகளை இயக்க
  • Flash திரைப்படங்களைக் காட்ட
  • வரைபடங்களைக் காட்ட
  • வைரஸ்களுக்காக சோதனை செய்ய
  • வங்கி அடையாளத்தைச் சரிபார்க்க

எச்சரிக்கை!

பெரும்பாலான உலாவிகள் இப்போது Java Applets மற்றும் Plug-ins ஆகியவற்றை ஆதரிப்பதில்லை.

ActiveX கட்டுப்பாடுகள் எந்த உலாவிகளிலும் ஆதரிக்கப்படுவதில்லை.

Shockwave Flash ஆதரவும் நவீன உலாவிகளில் முடக்கப்பட்டுள்ளது.

<object> உறுப்பு

<object> உறுப்பு அனைத்து உலாவிகளாலும் ஆதரிக்கப்படுகிறது.

<object> உறுப்பு ஒரு HTML ஆவணத்திற்குள் உட்பொதிக்கப்பட்ட பொருளை வரையறுக்கிறது.

இது வலைப்பக்கங்களில் சேர்க்கை நிரல்களை (Java applets, PDF readers, மற்றும் Flash Players போன்றவை) உட்பொதிப்பதற்காக வடிவமைக்கப்பட்டது, ஆனால் HTML ஐ HTML இல் சேர்க்கவும் பயன்படுத்தலாம்:

எடுத்துக்காட்டு

<object width="100%" height="500px" data="snippet.html"></object>

அல்லது நீங்கள் விரும்பினால் படங்களையும்:

எடுத்துக்காட்டு

<object data="audi.jpeg"></object>

<embed> உறுப்பு

<embed> உறுப்பு அனைத்து முக்கிய உலாவிகளாலும் ஆதரிக்கப்படுகிறது.

<embed> உறுப்பு ஒரு HTML ஆவணத்திற்குள் உட்பொதிக்கப்பட்ட பொருளை வரையறுக்கிறது.

வலை உலாவிகள் நீண்ட காலமாக <embed> உறுப்பை ஆதரித்து வருகின்றன. இருப்பினும், இது HTML5 க்கு முன் HTML வரையறையின் ஒரு பகுதியாக இல்லை.

எடுத்துக்காட்டு

<embed src="audi.jpeg">

குறிப்பு:

<embed> உறுப்புக்கு மூடும் குறிச்சொல் இல்லை. இது மாற்று உரையைக் கொண்டிருக்க முடியாது.

<embed> உறுப்பு HTML ஐ HTML இல் சேர்க்கவும் பயன்படுத்தலாம்:

எடுத்துக்காட்டு

<embed width="100%" height="500px" src="snippet.html">

<object> vs <embed>

<object> உறுப்பு

<object 
  data="file.pdf" 
  type="application/pdf" 
  width="100%" 
  height="500px">
  <p>Alternative text</p>
</object>

<embed> உறுப்பு

<embed 
  src="file.pdf" 
  type="application/pdf" 
  width="100%" 
  height="500px">

<object> நன்மைகள்

  • மூடும் குறிச்சொல் உண்டு
  • மாற்று உரையைக் கொண்டிருக்கலாம்
  • HTML தரநிலையின் பகுதி
  • அனைத்து உலாவிகளாலும் ஆதரிக்கப்படுகிறது

<embed> நன்மைகள்

  • எளிமையான தொடரியல்
  • மூடும் குறிச்சொல் தேவையில்லை
  • நீண்ட கால ஆதரவு
  • HTML5 இல் சேர்க்கப்பட்டது

நவீன மாற்றுகள்

பழைய சேர்க்கை நிரல்களுக்கான நவீன மாற்றுகள்:

Flash → HTML5 Video

Flash திரைப்படங்களுக்கு பதிலாக HTML5 <video> உறுப்பைப் பயன்படுத்தவும்.

Flash Audio → HTML5 Audio

Flash ஒலிக்கு பதிலாக HTML5 <audio> உறுப்பைப் பயன்படுத்தவும்.

ActiveX → JavaScript APIs

ActiveX கட்டுப்பாடுகளுக்கு பதிலாக நவீன JavaScript APIs ஐப் பயன்படுத்தவும்.

PDF Readers → Browser Support

PDF சேர்க்கை நிரல்களுக்கு பதிலாக உலாவியின் உள்ளமைக்கப்பட்ட PDF ஆதரவைப் பயன்படுத்தவும்.

अभ्यास

❓ பின்வரும் எந்த உறுப்பு சரியான <embed> தொடரியலைக் காட்டுகிறது?

<embed src="image.jpg"></embed>
✗ தவறு! <embed> உறுப்புக்கு மூடும் குறிச்சொல் தேவையில்லை
<embed src="image.jpg">
✓ சரி! <embed> உறுப்புக்கு மூடும் குறிச்சொல் தேவையில்லை
<embed><src="image.jpg"></embed>
✗ தவறு! <embed> உறுப்பில் மூடும் குறிச்சொல் மட்டும் இல்லை, ஆனால் src பண்புக்கூறு உள்ளே இருக்க வேண்டும்
