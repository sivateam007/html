HTML Audio

HTML <audio>तत्व का उपयोग वेब पेज पर ध्वनि फ़ाइल चलाने के लिए किया जाता है।

ध्वनि तत्व:

HTML में ध्वनि फ़ाइल चलाने के लिए,<audio>तत्व नियंत्रण के साथ प्रयोग किया जाता है। यह उपयोगकर्ताओं को प्ले, पॉज़ और वॉल्यूम जैसे नियंत्रण प्रदान करता है।

HTML <audio>अंग

HTML में ध्वनि फ़ाइल चलाने के लिए,<audio>तत्व का प्रयोग करें:

उदाहरण

<ऑडियो नियंत्रण>
  <स्रोत src='घोड़ा.ओग' प्रकार='ऑडियो/ओग'>
  <स्रोत src='घोड़ा.mp3' प्रकार='ऑडियो/mpeg'>
  आपका ब्राउज़र ध्वनि तत्व का समर्थन नहीं करता.
</ऑडियो>

HTML ध्वनि - यह कैसे काम करती है

controlsविशेषता नियंत्रण, ठहराव और वॉल्यूम जैसे ध्वनि नियंत्रण जोड़ता है।

<source>तत्व ब्राउज़र आपको वैकल्पिक ध्वनि फ़ाइलें निर्दिष्ट करने की अनुमति देता है जिन्हें चुना जा सकता है। ब्राउज़र पहले मान्यता प्राप्त प्रारूप का उपयोग करेगा.

<audio>और</audio>टैग के बीच पाठ<audio>केवल उन ब्राउज़रों में प्रदर्शित होता है जो तत्व का समर्थन नहीं करते हैं।

HTML <audio>स्वचालित गति

ऑडियो फ़ाइल को स्वचालित रूप से प्रारंभ करने के लिए,autoplayविशेषता का प्रयोग करें:

उदाहरण

<ऑडियो ऑटोप्ले को नियंत्रित करता है>
  <स्रोत src='घोड़ा.ओग' प्रकार='ऑडियो/ओग'>
  <स्रोत src='घोड़ा.mp3' प्रकार='ऑडियो/mpeg'>
  आपका ब्राउज़र ध्वनि तत्व का समर्थन नहीं करता.
</ऑडियो>

टिप्पणी:

क्रोमियम ब्राउज़र अधिकांश मामलों में स्वचालित सक्रियण की अनुमति नहीं देते हैं। हालाँकि, अक्षम स्वचालित संचालन की हमेशा अनुमति होती है।

अपनी ध्वनि फ़ाइल को स्वचालित रूप से चलाएं (लेकिन अक्षम)।autoplayबादmutedजोड़ना:

उदाहरण

<ऑडियो नियंत्रण ऑटोप्ले म्यूट किया गया>
  <स्रोत src='घोड़ा.ओग' प्रकार='ऑडियो/ओग'>
  <स्रोत src='घोड़ा.mp3' प्रकार='ऑडियो/mpeg'>
  आपका ब्राउज़र ध्वनि तत्व का समर्थन नहीं करता.
</ऑडियो>

HTML ध्वनि - विधियाँ, गुण और घटनाएँ

HTML DOM <audio>तत्व के लिए तरीकों, गुणों और घटनाओं को परिभाषित करता है।

यह आपको ध्वनियों को लोड करने, चलाने और रोकने और अवधि और वॉल्यूम निर्धारित करने की अनुमति देता है।

ऐसे DOM ईवेंट भी हैं जो ध्वनि बजने, रुकने आदि पर आपको सूचित कर सकते हैं।

जावास्क्रिप्ट कोड उदाहरण:

// ध्वनि तत्व प्राप्त करें
var ऑडियो = document.getElementById('demoAudio');

// ध्वनि बजाओ
फ़ंक्शन प्लेऑडियो() {
    ऑडियो.प्ले();
}

// ध्वनि रोकें
फ़ंक्शन पॉज़ऑडियो() {
    ऑडियो.रोकें();
}

// वॉल्यूम बढ़ाएँ
फ़ंक्शन वॉल्यूमअप() {
    यदि(ऑडियो.वॉल्यूम <1.0) {
        ऑडियो.वॉल्यूम += 0.1;
    }
}

// वॉल्यूम कम करें
फ़ंक्शन वॉल्यूमडाउन() {
    यदि(ऑडियो.वॉल्यूम > 0.0) {
        ऑडियो.वॉल्यूम -= 0.1;
    }
}

// ध्वनि फिर से शुरू करें
फ़ंक्शन पुनरारंभऑडियो() {
    ऑडियो.वर्तमान समय = 0;
    ऑडियो.प्ले();
}

अभ्यास

❓निम्नलिखित में से कौन सा HTML कोड सही ध्वनि तत्व प्रदर्शित करता है?

<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
</audio>
✓ ठीक है! यह कोड ध्वनि तत्व को सही नियंत्रण के साथ दिखाता है
<audio>
  <src="music.mp3">
</audio>
✗ ग़लत! <src> एक मान्य HTML टैग नहीं है. <स्रोत> का उपयोग किया जाना चाहिए
<sound controls>
  <source src="music.mp3">
</sound>
✗ ग़लत! <ध्वनि> एक मान्य HTML टैग नहीं है. <ऑडियो> का उपयोग किया जाना चाहिए
