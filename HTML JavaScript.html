The HTML <script> Tag

HTML <script> குறிச்சொல் ஒரு கிளையண்ட்-சைட் ஸ்கிரிப்டை (JavaScript) வரையறுக்கப் பயன்படுகிறது.

<script> உறுப்பு ஸ்கிரிப்ட் அறிக்கைகளைக் கொண்டிருக்கலாம், அல்லது src பண்புக்கூறு மூலம் வெளி ஸ்கிரிப்ட் கோப்பைச் சுட்டிக்காட்டலாம்.

JavaScript க்கான பொதுவான பயன்பாடுகள் பட மேலாண்மை, படிவ சரிபார்ப்பு மற்றும் உள்ளடக்கத்தின் டைனமிக் மாற்றங்கள் ஆகும்.

ஒரு HTML உறுப்பைத் தேர்ந்தெடுக்க, JavaScript பெரும்பாலும் document.getElementById() முறையைப் பயன்படுத்துகிறது.

இந்த JavaScript எடுத்துக்காட்டு "Hello JavaScript!" என்று id="demo" உள்ள HTML உறுப்புக்குள் எழுதுகிறது:

Example

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

உதவிக்குறிப்பு:

எங்கள் JavaScript டுடோரியலில் JavaScript பற்றி மேலும் அறியலாம்.

A Taste of JavaScript

JavaScript என்ன செய்ய முடியும் என்பதற்கு சில எடுத்துக்காட்டுகள் இங்கே:

Example

JavaScript உள்ளடக்கத்தை மாற்றலாம்:

document.getElementById("demo").innerHTML = "Hello JavaScript!";

Example

JavaScript ஸ்டைல்களை மாற்றலாம்:

document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";

Example

JavaScript பண்புக்கூறுகளை மாற்றலாம்:

document.getElementById("image").src = "picture.gif";

The HTML <noscript> Tag

HTML <noscript> குறிச்சொல் அவர்களின் உலாவியில் ஸ்கிரிப்ட்களை முடக்கிய பயனர்களுக்கு அல்லது ஸ்கிரிப்ட்களை ஆதரிக்காத உலாவியைக் கொண்ட பயனர்களுக்கு காட்ட வேண்டிய மாற்று உள்ளடக்கத்தை வரையறுக்கிறது:

Example

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
<noscript>Sorry, your browser does not support JavaScript!</noscript>
