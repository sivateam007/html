YouTube காணொளிகளை வலைப்பக்கங்களில் உட்பொதித்தல்

HTML YouTube Videos

HTML இல் காணொளிகளை இயக்க எளிய வழி, YouTube ஐப் பயன்படுத்துவதாகும்.

காணொளி வடிவங்களில் சிக்கலா?

காணொளிகளை வெவ்வேறு வடிவங்களாக மாற்றுவது கடினமானது மற்றும் நேரம் எடுக்கக்கூடியது.

எளிமையான தீர்வு YouTube உங்கள் வலைப்பக்கத்தில் காணொளிகளை இயக்க விடுவதாகும்.

YouTube காணொளி அடையாளம்

நீங்கள் ஒரு காணொளியை சேமிக்கும்போது (அல்லது இயக்கும்போது), YouTube ஒரு அடையாளத்தை (tgbNymZ7vqY போன்றது) காட்டும்.

இந்த அடையாளத்தைப் பயன்படுத்தி, உங்கள் காணொளியை HTML குறியீட்டில் குறிப்பிடலாம்.

YouTube Video ID: tgbNymZ7vqY
YouTube URL: https://www.youtube.com/watch?v=tgbNymZ7vqY
Embed URL: https://www.youtube.com/embed/tgbNymZ7vqY

HTML இல் YouTube காணொளியை இயக்குதல்

உங்கள் காணொளியை ஒரு வலைப்பக்கத்தில் இயக்க, பின்வரும் படிகளைப் பின்பற்றவும்:

  1. காணொளியை YouTube இல் பதிவேற்றவும்
  2. காணொளி அடையாளத்தைக் குறிப்பெடுக்கவும்
  3. உங்கள் வலைப்பக்கத்தில் ஒரு <iframe> உறுப்பை வரையறுக்கவும்
  4. src பண்புக்கூறு காணொளி URL ஐ சுட்டிக்காட்டட்டும்
  5. இயக்கியின் பரிமாணத்தைக் குறிப்பிட width மற்றும் height பண்புக்கூறுகளைப் பயன்படுத்தவும்
  6. URL க்கு வேறு அளவுருக்களைச் சேர்க்கவும் (கீழே காண்க)

எடுத்துக்காட்டு

<iframe width="420" height="315"
src="https://www.youtube.com/embed/tgbNymZ7vqY">
</iframe>

YouTube தானியங்கி இயக்கம் + முடக்கு

YouTube URL க்கு autoplay=1 ஐச் சேர்ப்பதன் மூலம், ஒரு பயனர் பக்கத்தைப் பார்வையிடும் போது உங்கள் காணொளி தானாக இயங்கத் தொடங்கும். இருப்பினும், தானாக காணொளியைத் தொடங்குவது உங்கள் பார்வையாளர்களுக்கு எரிச்சலூட்டும்!

குறிப்பு:

குரோமியம் உலாவிகள் பெரும்பாலான நிகழ்வுகளில் தானியங்கி இயக்கத்தை அனுமதிப்பதில்லை. இருப்பினும், முடக்கப்பட்ட தானியங்கி இயக்கம் எப்போதும் அனுமதிக்கப்படுகிறது.

உங்கள் காணொளி தானாக இயங்குவதற்கு (ஆனால் முடக்கப்பட்டது) autoplay=1 க்குப் பிறகு mute=1 ஐச் சேர்க்கவும்.

YouTube - தானியங்கி இயக்கம் + முடக்கு

<iframe width="420" height="315"
src="https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=1&mute=1">
</iframe>

YouTube பட்டியல்

இயக்க வேண்டிய காணொளிகளின் காற்புள்ளி பிரிக்கப்பட்ட பட்டியல் (அசல் URL க்கு கூடுதலாக).

YouTube மீள் சுழற்சி

உங்கள் காணொளி என்றென்றும் மீண்டும் மீண்டும் இயங்குவதற்கு playlist=videoID மற்றும் loop=1 ஐச் சேர்க்கவும்.

  • loop=0 (இயல்புநிலை) - காணொளி ஒருமுறை மட்டுமே இயங்கும்.
  • loop=1 - காணொளி மீண்டும் மீண்டும் இயங்கும் (என்றென்றும்).

YouTube - என்றென்றும் மீள் சுழற்சி

<iframe width="420" height="315"
src="https://www.youtube.com/embed/tgbNymZ7vqY?playlist=tgbNymZ7vqY&loop=1">
</iframe>

YouTube கட்டுப்பாடுகள்

காணொளி இயக்கியில் கட்டுப்பாடுகளைக் காட்டாமல் இருக்க controls=0 ஐச் சேர்க்கவும்.

  • controls=0 - இயக்கி கட்டுப்பாடுகள் காட்டப்படாது.
  • controls=1 (இயல்புநிலை) - இயக்கி கட்டுப்பாடுகள் காட்டப்படும்.

YouTube - கட்டுப்பாடுகள்

<iframe width="420" height="315"
src="https://www.youtube.com/embed/tgbNymZ7vqY?controls=0">
</iframe>

YouTube URL அளவுருக்கள்

autoplay

காணொளியை தானாக இயக்க

0 - இல்லை (இயல்புநிலை)

1 - ஆம்

mute

காணொளியை முடக்கு

0 - இல்லை (இயல்புநிலை)

1 - ஆம்

controls

இயக்கி கட்டுப்பாடுகளைக் காட்டு

0 - மறை

1 - காட்டு (இயல்புநிலை)

loop

காணொளியை மீண்டும் இயக்கு

0 - ஒருமுறை (இயல்புநிலை)

1 - மீள் சுழற்சி

rel

தொடர்புடைய காணொளிகளைக் காட்டு

0 - மறை

1 - காட்டு (இயல்புநிலை)

modestbranding

YouTube லோகோவைக் குறை

0 - இயல்புநிலை

1 - குறைந்த லோகோ

URL அளவுருக்களை இணைத்தல்:

பல அளவுருக்களை இணைக்க, & சின்னத்தைப் பயன்படுத்தவும்:

?autoplay=1&mute=1&controls=0&loop=1

முழுமையான எடுத்துக்காட்டு

YouTube காணொளி உட்பொதிப்பு - முழு எடுத்துக்காட்டு

<!DOCTYPE html>
<html>
<head>
    <title>YouTube Video Example - Jassif Team</title>
</head>
<body>

<h1>HTML YouTube Video Embedding</h1>

<!-- YouTube Video with Parameters -->
<iframe 
    width="560" 
    height="315" 
    src="https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=0&mute=0&controls=1&loop=0" 
    title="YouTube video player" 
    frameborder="0" 
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
    allowfullscreen>
</iframe>

</body>
</html>

अभ्यास

❓ பின்வரும் எந்த YouTube URL காணொளியை தானாக இயக்கும் ஆனால் ஒலியில்லாமல்?

https://www.youtube.com/embed/VIDEO_ID?autoplay=1
✗ தவறு! இந்த URL காணொளியை தானாக இயக்கும், ஆனால் ஒலியுடன் இருக்கும்
https://www.youtube.com/embed/VIDEO_ID?autoplay=1&mute=1
✓ சரி! autoplay=1 மற்றும் mute=1 இரண்டும் தேவைப்படுகின்றன
https://www.youtube.com/embed/VIDEO_ID?play=1&silent=1
✗ தவறு! "play" மற்றும் "silent" என்பவை செல்லுபடியாகும் YouTube URL அளவுருக்கள் அல்ல
