HTML Iframes

एक वेब पेज के भीतर एक वेब पेज प्रदर्शित करने के लिए HTML iframe का उपयोग किया जाता है।

उपरोक्त iframe HTML Tutorial.html पृष्ठ प्रदर्शित करता है

HTML Iframe Syntax

HTML <iframe> टैग एक इनलाइन फ़्रेम निर्दिष्ट करता है।

वर्तमान HTML दस्तावेज़ में किसी अन्य दस्तावेज़ को एम्बेड करने के लिए एक इनलाइन फ़्रेम का उपयोग किया जाता है।

Syntax

<iframe src="url" title="description"></iframe>

बख्शीश:

<iframe> में हमेशा एक शीर्षक विशेषता जोड़ना अच्छा अभ्यास है। इसका उपयोग स्क्रीन रीडर्स द्वारा यह पढ़ने के लिए किया जाता है कि आईफ्रेम की सामग्री क्या है।

Iframe - Set Height and Width

आईफ्रेम का आकार निर्दिष्ट करने के लिए ऊंचाई और चौड़ाई विशेषताओं का उपयोग करें।

ऊँचाई और चौड़ाई डिफ़ॉल्ट रूप से पिक्सेल में निर्दिष्ट होती है:

Example

<iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Example"></iframe>

या आप शैली विशेषता जोड़ सकते हैं और सीएसएस ऊंचाई और चौड़ाई विशेषताओं का उपयोग कर सकते हैं:

Example

<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Iframe Example"></iframe>

Iframe - Remove the Border

डिफ़ॉल्ट रूप से, एक iframe के चारों ओर एक बॉर्डर होता है।

बॉर्डर हटाने के लिए, स्टाइल विशेषता जोड़ें और CSS बॉर्डर विशेषता का उपयोग करें:

Example

<iframe src="demo_iframe.htm" style="border:none;" title="Iframe Example"></iframe>

CSS के साथ, आप iframe के बॉर्डर का आकार, शैली और रंग भी बदल सकते हैं:

Example

<iframe src="demo_iframe.htm" style="border:2px solid red;" title="Iframe Example"></iframe>

Iframe - Target for a Link

एक आईफ्रेम का उपयोग किसी लिंक के लिए लक्ष्य फ्रेम के रूप में किया जा सकता है।

लिंक की लक्ष्य विशेषता को आईफ्रेम की नाम विशेषता को संदर्भित करना चाहिए:

Example

<iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe>

<p><a href="https://www.jassifteam.com" target="iframe_a">JassifTeam.com</a></p>

Chapter Summary

  • HTML <iframe> टैग एक इनलाइन फ़्रेम निर्दिष्ट करता है
  • Src विशेषता एम्बेड किए जाने वाले पृष्ठ के URL को परिभाषित करती है
  • हमेशा शीर्षक विशेषता शामिल करें (स्क्रीन रीडर के लिए)
  • ऊँचाई और चौड़ाई विशेषताएँ iframe का आकार निर्दिष्ट करती हैं
  • iframe border:none के चारों ओर का बॉर्डर हटाने के लिए; उपयोग

Exercise

❓ In which IFRAME attribute can we specify an URL?

href
✗ ग़लत! href विशेषता का उपयोग लिंक के लिए किया जाता है, iframe के लिए नहीं।
src
✓ ठीक है! Src विशेषता में हम iframe में प्रदर्शित करने के लिए पृष्ठ का URL निर्दिष्ट करते हैं।
url
✗ ग़लत! यूआरएल HTML में मान्य विशेषता नहीं है. आईफ्रेम के लिए src का उपयोग किया जाता है।

Tag Description
