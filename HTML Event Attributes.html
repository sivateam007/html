HTML Event Attributes

HTML நிகழ்வு பண்புகள்

Global Event Attributes

HTML உலாவியில் நிகழ்வுகள் செயல்களைத் தூண்டும் திறனைக் கொண்டுள்ளது, பயனர் ஒரு உறுப்பில் கிளிக் செய்யும் போது JavaScript தொடங்குவது போன்றது.

நிகழ்வுகளை நிரலாக்கம் பற்றி மேலும் அறிய, எங்கள் JavaScript டுடோரியலைப் பார்வையிடவும்.

கீழே உள்ளவை நிகழ்வு செயல்களை வரையறுக்க HTML கூறுகளில் சேர்க்கக்கூடிய உலகளாவிய நிகழ்வு பண்புகளாகும்.

Window Event Attributes

சாளர பொருளுக்கான நிகழ்வுகள் (<body> குறிச்சொல்லுக்குப் பொருந்தும்):

பண்பு மதிப்பு விளக்கம்
onafterprint script ஆவணம் அச்சிடப்பட்ட பிறகு இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onbeforeprint script ஆவணம் அச்சிடப்படுவதற்கு முன் இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onbeforeunload script ஆவணம் இறக்கப்பட இருக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onerror script பிழை ஏற்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onhashchange script ஒரு URL இன் நங்கூரப் பகுதியில் மாற்றங்கள் ஏற்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onload script பக்கம் ஏற்றுதல் முடிந்ததும் தூண்டப்படும்
onmessage script செய்தி தூண்டப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onoffline script உலாவி ஆஃப்லைனில் வேலை செய்யத் தொடங்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ononline script உலாவி ஆன்லைனில் வேலை செய்யத் தொடங்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onpagehide script பயனர் ஒரு பக்கத்திலிருந்து வெளியேறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onpageshow script பயனர் ஒரு பக்கத்திற்கு செல்லும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onpopstate script சாளரத்தின் வரலாறு மாறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onresize script உலாவி சாளரம் அளவிடப்படும் போது தூண்டப்படும்
onstorage script ஒரு வெப் ஸ்டோரேஜ் பகுதி புதுப்பிக்கப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onunload script ஒரு பக்கம் இறக்கப்பட்ட பிறகு தூண்டப்படும் (அல்லது உலாவி சாளரம் மூடப்பட்ட பிறகு)

Form Events

HTML படிவத்திற்குள் செயல்களால் தூண்டப்படும் நிகழ்வுகள் (அனைத்து HTML கூறுகளுக்கும் பொருந்தும், ஆனால் படிவ கூறுகளில் அதிகம் பயன்படுத்தப்படுகிறது):

பண்பு மதிப்பு விளக்கம்
onblur script கூறு கவனத்தை இழக்கும் தருணத்தில் தூண்டப்படும்
onchange script கூறின் மதிப்பு மாறும் தருணத்தில் தூண்டப்படும்
oncontextmenu script ஒரு சூழல் மெனு தூண்டப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onfocus script கூறு கவனம் பெறும் தருணத்தில் தூண்டப்படும்
oninput script ஒரு கூறு பயனர் உள்ளீட்டைப் பெறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
oninvalid script ஒரு கூறு செல்லாததாக இருக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onreset script படிவத்தில் உள்ள மீட்டமை பொத்தான் கிளிக் செய்யப்படும் போது தூண்டப்படும்
onsearch script பயனர் ஒரு தேடல் புலத்தில் ஏதாவது எழுதும் போது தூண்டப்படும் (<input="search">க்கு)
onselect script ஒரு கூறில் சில உரை தேர்ந்தெடுக்கப்பட்ட பிறகு தூண்டப்படும்
onsubmit script படிவம் சமர்ப்பிக்கப்படும் போது தூண்டப்படும்

Keyboard Events

பண்பு மதிப்பு விளக்கம்
onkeydown script பயனர் ஒரு விசையை அழுத்தும் போது தூண்டப்படும்
onkeypress script பயனர் ஒரு விசையை அழுத்தும் போது தூண்டப்படும்
onkeyup script பயனர் ஒரு விசையை விடும் போது தூண்டப்படும்

Mouse Events

பண்பு மதிப்பு விளக்கம்
onclick script கூறில் சுட்டி கிளிக் செய்யும் போது தூண்டப்படும்
ondblclick script கூறில் சுட்டி இரட்டைக் கிளிக் செய்யும் போது தூண்டப்படும்
onmousedown script ஒரு சுட்டி பொத்தான் ஒரு கூறில் அழுத்தப்படும் போது தூண்டப்படும்
onmousemove script சுட்டி சுட்டி ஒரு கூறின் மேல் நகரும் போது தூண்டப்படும்
onmouseout script சுட்டி சுட்டி ஒரு கூறை விட்டு வெளியேறும் போது தூண்டப்படும்
onmouseover script சுட்டி சுட்டி ஒரு கூறின் மேல் நகரும் போது தூண்டப்படும்
onmouseup script ஒரு சுட்டி பொத்தான் ஒரு கூறின் மேல் விடப்படும் போது தூண்டப்படும்
onmousewheel script காலாவதியானது. onwheel பண்பைப் பயன்படுத்தவும்
onwheel script சுட்டி சக்கரம் மேலே அல்லது கீழே உருளும் போது தூண்டப்படும்

Drag Events

பண்பு மதிப்பு விளக்கம்
ondrag script ஒரு கூறு இழுக்கப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondragend script இழுத்தல் செயல்பாடு முடிவடையும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondragenter script ஒரு கூறு செல்லுபடியாகும் விடுபடும் இலக்குக்கு இழுக்கப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondragleave script ஒரு கூறு செல்லுபடியாகும் விடுபடும் இலக்கை விட்டு வெளியேறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondragover script ஒரு கூறு செல்லுபடியாகும் விடுபடும் இலக்கின் மேல் இழுக்கப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondragstart script இழுத்தல் செயல்பாடு தொடங்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondrop script இழுக்கப்பட்ட கூறு விடப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onscroll script ஒரு கூறின் ஸ்க்ரோல்பார் ஸ்க்ரோல் செய்யப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்

Clipboard Events

பண்பு மதிப்பு விளக்கம்
oncopy script பயனர் ஒரு கூறின் உள்ளடக்கத்தை நகலெடுக்கும் போது தூண்டப்படும்
oncut script பயனர் ஒரு கூறின் உள்ளடக்கத்தை வெட்டும் போது தூண்டப்படும்
onpaste script பயனர் ஒரு கூறில் சில உள்ளடக்கங்களை ஒட்டும் போது தூண்டப்படும்

Media Events

வீடியோக்கள், படங்கள் மற்றும் ஆடியோ போன்ற ஊடகங்களால் தூண்டப்படும் நிகழ்வுகள் (அனைத்து HTML கூறுகளுக்கும் பொருந்தும், ஆனால் <audio>, <embed>, <img>, <object>, மற்றும் <video> போன்ற ஊடக கூறுகளில் பொதுவானது).

பண்பு மதிப்பு விளக்கம்
onabort script நிறுத்தப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
oncanplay script ஒரு கோப்பு இயங்கத் தொடங்கத் தயாராக இருக்கும்போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் (தொடங்குவதற்கு போதுமானது தாங்கும் போது)
oncanplaythrough script ஒரு கோப்பு தாங்குவதை நிறுத்தாமல் இறுதிவரை இயக்கப்பட முடியும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
oncuechange script <track> கூறில் கியூ மாறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ondurationchange script ஊடகத்தின் நீளம் மாறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onemptied script ஏதேனும் மோசமான நிகழ்வு ஏற்பட்டு கோப்பு திடீரென கிடைக்காமல் போகும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் (எதிர்பாராத விதமாக துண்டிக்கப்பட்டது போன்றது)
onended script ஊடகம் முடிவை அடையும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் ("கேட்டதற்கு நன்றி" போன்ற செய்திகளுக்கு பயனுள்ள நிகழ்வு)
onerror script கோப்பு ஏற்றப்படும் போது பிழை ஏற்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onloadeddata script ஊடகத் தரவு ஏற்றப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onloadedmetadata script மெட்டாடேட்டா (பரிமாணங்கள் மற்றும் अवधि போன்றவை) ஏற்றப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onloadstart script எதுவும் உண்மையில் ஏற்றப்படுவதற்கு முன் கோப்பு ஏற்றத் தொடங்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onpause script ஊடகம் பயனரால் அல்லது நிரலாக்கத்தால் நிறுத்தப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onplay script ஊடகம் இயங்கத் தொடங்கத் தயாராக இருக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onplaying script ஊடகம் உண்மையில் இயங்கத் தொடங்கியது என்பதைக் குறிக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onprogress script உலாவி ஊடகத் தரவைப் பெறும் செயல்பாட்டில் இருக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onratechange script பிளேபேக் விகிதம் மாறும் ஒவ்வொரு முறையும் இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் (பயனர் மெதுநிலை அல்லது வேகமுன்னேற்ற முறைக்கு மாறும் போது போன்றது)
onseeked script தேடுதல் பண்பு தவறானது என அமைக்கப்பட்டால், தேடுதல் முடிவடைந்தது என்பதைக் குறிக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onseeking script தேடுதல் பண்பு உண்மை என அமைக்கப்பட்டால், தேடுதல் செயலில் உள்ளது என்பதைக் குறிக்கும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onstalled script உலாவி எந்தக் காரணத்திற்காகவும் ஊடகத் தரவைப் பெற முடியாதபோது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
onsuspend script ஊடகத் தரவு முழுமையாக ஏற்றப்படுவதற்கு முன், எந்தக் காரணத்திற்காகவும் பெறுவது நிறுத்தப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட்
ontimeupdate script இயக்கும் நிலை மாறும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் (பயனர் ஊடகத்தில் வேறுபட்ட புள்ளிக்கு வேகமுன்னேற்றம் செய்யும் போது போன்றது)
onvolumechange script அளவு மாறும் ஒவ்வொரு முறையும் இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் ("முடக்கு" என அளவை அமைப்பதையும் இது உள்ளடக்கியது)
onwaiting script ஊடகம் நிறுத்தப்பட்டுள்ளது ஆனால் மீண்டும் தொடங்கும் என்று எதிர்பார்க்கப்படும் போது இயக்கப்பட வேண்டிய ஸ்கிரிப்ட் (ஊடகம் மேலும் தாங்குவதற்காக நிறுத்தப்படும் போது போன்றது)
