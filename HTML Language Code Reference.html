ISO மொழிக் குறியீடுகள்

வலைப்பக்கத்தின் மொழியை அறிவிக்க, குறிச்சொல்லுக்குள் lang பண்பை எப்போதும் சேர்க்க வேண்டும். இது தேடுபொறிகள் மற்றும் உலாவிகளுக்கு உதவுவதற்காக உள்ளது:

<html lang="en">
...
</html>

XHTML இல், மொழி குறிச்சொல்லுக்குள் பின்வருமாறு அறிவிக்கப்படுகிறது:

<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
...
</html>

ISO 639-1 மொழிக் குறியீடுகள்

ISO 639-1 மொழிகளுக்கான சுருக்கங்களை வரையறுக்கிறது:

மேலும் பார்க்க: நாட்டுக் குறியீடுகளுக்கான குறிப்பு.
