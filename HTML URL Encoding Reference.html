URL குறியாக்க குறிப்பு

URL - யுனிஃபார்ம் ரிசோர்ஸ் லொகேட்டர்

வலை உலாவிகள் வலை சேவையகங்களிடமிருந்து பக்கங்களைக் கோருவதற்கு URL ஐப் பயன்படுத்துகின்றன.

URL என்பது ஒரு வலைப்பக்கத்தின் முகவரி, எடுத்துக்காட்டாக: https://www.jassifteam.com.

URL குறியாக்கம் (சதவீத குறியாக்கம்)

URL குறியாக்கம் எழுத்துகளை இணையம் வழியாக பரிமாற்றம் செய்யக்கூடிய வடிவத்திற்கு மாற்றுகிறது.

URLகளை இணையம் வழியாக அனுப்புவதற்கு ASCII எழுத்துத்தொகுப்பை மட்டுமே பயன்படுத்த முடியும்.

URLகள் பெரும்பாலும் ASCII தொகுப்பிற்கு வெளியே உள்ள எழுத்துகளைக் கொண்டிருப்பதால், URL செல்லுபடியாகும் ASCII வடிவத்திற்கு மாற்றப்பட வேண்டும்.

URL குறியாக்கம் பாதுகாப்பற்ற ASCII எழுத்துகளை "%" க்குப் பிறகு இரண்டு ஹெக்ஸாடெசிமல் இலக்கங்களுடன் மாற்றுகிறது.

URLகளில் இடைவெளிகள் இருக்க முடியாது. URL குறியாக்கம் பொதுவாக இடைவெளியை கூட்டல் (+) அடையாளத்துடன் அல்லது %20 உடன் மாற்றுகிறது.

சுயமாக முயற்சிக்கவும்

கீழே உள்ள "சமர்ப்பிக்கவும்" பொத்தானைக் கிளிக் செய்தால், உள்ளீடு சேவையகத்திற்கு அனுப்பப்படுவதற்கு முன் உலாவி URL குறியாக்கம் செய்யும். சேவையகத்தில் உள்ள ஒரு பக்கம் பெறப்பட்ட உள்ளீட்டைக் காண்பிக்கும்.

Hello Günter

வேறு சில உள்ளீடுகளை முயற்சித்து மீண்டும் சமர்ப்பிக்கவும்.

URL குறியாக்க செயல்பாடுகள்

ஜாவாஸ்கிரிப்ட், PHP மற்றும் ASP ஆகியவற்றில் ஒரு சரத்தை URL குறியாக்கம் செய்ய பயன்படுத்தக்கூடிய செயல்பாடுகள் உள்ளன.

PHP இல் rawurlencode() செயல்பாடு உள்ளது, மேலும் ASP இல் Server.URLEncode() செயல்பாடு உள்ளது.

ஜாவாஸ்கிரிப்டில் நீங்கள் encodeURIComponent() செயல்பாட்டைப் பயன்படுத்தலாம்.

ஜாவாஸ்கிரிப்ட் செயல்பாடு உரையை எவ்வாறு குறியாக்கம் செய்கிறது என்பதைப் பார்க்க "URL குறியாக்கம்" பொத்தானைக் கிளிக் செய்யவும்.

Hello Günter

குறிப்பு:

ஜாவாஸ்கிரிப்ட் செயல்பாடு இடைவெளியை %20 ஆக குறியாக்கம் செய்கிறது.
