HTML style பண்புக்கூறு

HTML உறுப்பின் ஸ்டைலை அமைப்பது style பண்புக்கூறு மூலம் செய்ய முடியும்.

HTML style பண்புக்கூறு பின்வரும் தொடரியலைக் கொண்டுள்ளது:

<tagname style="property:value;">

property என்பது ஒரு CSS பண்பாகும். value என்பது ஒரு CSS மதிப்பாகும்.

CSS பற்றி இந்த டுடோரியலில் பின்னர் மேலும் கற்றுக்கொள்வீர்கள்.

பின்னணி நிறம்

CSS background-color பண்பு ஒரு HTML உறுப்புக்கான பின்னணி நிறத்தை வரையறுக்கிறது.

எடுத்துக்காட்டு

பக்கத்தின் பின்னணி நிறத்தை powderblue ஆக அமைக்கவும்:

<body style="background-color:powderblue;">

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>

எடுத்துக்காட்டு

இரண்டு வெவ்வேறு உறுப்புகளுக்கான பின்னணி நிறத்தை அமைக்கவும்:

<body>

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

</body>

உரை நிறம்

CSS color பண்பு ஒரு HTML உறுப்புக்கான உரை நிறத்தை வரையறுக்கிறது:

எடுத்துக்காட்டு

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

எழுத்துருக்கள்

CSS font-family பண்பு ஒரு HTML உறுப்புக்குப் பயன்படுத்தப்பட வேண்டிய எழுத்துருவை வரையறுக்கிறது:

எடுத்துக்காட்டு

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

உரை அளவு

CSS font-size பண்பு ஒரு HTML உறுப்புக்கான உரை அளவை வரையறுக்கிறது:

எடுத்துக்காட்டு

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

உரை சீரமைப்பு

CSS text-align பண்பு ஒரு HTML உறுப்புக்கான கிடைமட்ட உரை சீரமைப்பை வரையறுக்கிறது:

எடுத்துக்காட்டு

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

அத்தியாய சுருக்கம்

  • HTML உறுப்புகளை ஸ்டைல் செய்ய style பண்புக்கூறைப் பயன்படுத்தவும்
  • பின்னணி நிறத்திற்கு background-color பயன்படுத்தவும்
  • உரை நிறங்களுக்கு color பயன்படுத்தவும்
  • உரை எழுத்துருக்களுக்கு font-family பயன்படுத்தவும்
  • உரை அளவுகளுக்கு font-size பயன்படுத்தவும்
  • உரை சீரமைப்புக்கு text-align பயன்படுத்தவும்

अभ्यासகள் மூலம் கற்றல்

❓ HTML ஆவணத்திற்கு பின்னணி நிறத்தைச் சேர்ப்பதற்கான சரியான வழி எது?

<body style='background-color:pink;'>
✓ சரி! இது HTML இல் பின்னணி நிறத்தை அமைக்க சரியான தொடரியல்
<body background-color='pink;'>
✗ தவறு! background-color என்பது CSS பண்பு, HTML பண்புக்கூறு அல்ல
<body style='bgcolor:pink;'>
✗ தவறு! bgcolor என்பது பழைய HTML பண்புக்கூறு, CSS இல் background-color பயன்படுத்தவும்
