HTML Computer Code Elements

HTML में उपयोगकर्ता इनपुट और कंप्यूटर कोड को परिभाषित करने के लिए कई तत्व शामिल हैं।

Example

HTML Code:

<code>
x = 5;
y = 6;
z = x + y;
</code>

Output:

x = 5; y = 6; z = x + y;

HTML <kbd> For Keyboard Input

HTML <kbd>एलिमेंट का उपयोग कीबोर्ड इनपुट को परिभाषित करने के लिए किया जाता है। अंदर की सामग्री ब्राउज़र के डिफ़ॉल्ट मोनोस्पेस फ़ॉन्ट में प्रदर्शित होती है।

Example

किसी दस्तावेज़ में कुछ टेक्स्ट को कीबोर्ड इनपुट के रूप में परिभाषित करें:

HTML Code:

<p>Save the document by pressing <kbd>Ctrl + S</kbd></p>

Result:

Save the document by pressing Ctrl + S

उपयोग नोट:

<kbd>कीबोर्ड कमांड, शॉर्टकट और उपयोगकर्ता इनपुट प्रदर्शित करने के लिए तत्व का उपयोग करें।

HTML <samp> For Program Output

HTML <samp>एलिमेंट का उपयोग कंप्यूटर प्रोग्राम से नमूना आउटपुट को परिभाषित करने के लिए किया जाता है। अंदर की सामग्री ब्राउज़र के डिफ़ॉल्ट मोनोस्पेस फ़ॉन्ट में प्रदर्शित होती है।

Example

किसी दस्तावेज़ में कुछ पाठ को कंप्यूटर प्रोग्राम से नमूना आउटपुट के रूप में परिभाषित करें:

HTML Code:

<p>Message from my computer:</p>
<p><samp>File not found.<br>Press F1 to continue</samp></p>

Result:

Message from my computer:

File not found.
Press F1 to continue

HTML <code> For Computer Code

HTML <code>तत्व का उपयोग कंप्यूटर कोड को परिभाषित करने के लिए किया जाता है। अंदर की सामग्री ब्राउज़र के डिफ़ॉल्ट मोनोस्पेस फ़ॉन्ट में प्रदर्शित होती है।

Example

किसी दस्तावेज़ में कुछ टेक्स्ट को कंप्यूटर कोड के रूप में परिभाषित करें:

HTML Code:

<code>
x = 5;
y = 6;
z = x + y;
</code>

Result:

x = 5; y = 6; z = x + y;

टिप्पणी:

<code>तत्व अतिरिक्त सफेद स्थान और लाइन-ब्रेक को संरक्षित नहीं करता है।

Preserve Line-Breaks

अतिरिक्त सफेद स्थान और लाइन-ब्रेक को संरक्षित करने के लिए,<code>अंग<pre>तत्व के अंदर रखा जा सकता है:

Example

HTML Code:

<pre>
<code>
x = 5;
y = 6;
z = x + y;
</code>
</pre>

Result:

x = 5;
y = 6;
z = x + y;

सीखना:

<pre>तत्व पूर्वनिर्धारित पाठ को परिभाषित करता है जो फ़ॉन्ट और सफेद स्थान के सभी स्वरूपण को संरक्षित करता है।

HTML <var> For Variables

HTML <var>एलिमेंट का उपयोग प्रोग्रामिंग या गणितीय अभिव्यक्ति में एक चर को परिभाषित करने के लिए किया जाता है। अंदर की सामग्री आमतौर पर इटैलिक में प्रदर्शित होती है।

Example

किसी दस्तावेज़ में कुछ पाठ को चर के रूप में परिभाषित करें:

HTML Code:

<p>The area of a triangle is: 1/2 x <var>b</var> x <var>h</var>, where <var>b</var> is the base, and <var>h</var> is the vertical height.</p>

Result:

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.
