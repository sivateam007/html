HTML ஆவண வகை அறிவிப்பு பற்றி அறிக

HTML ஆவண வகை

அனைத்து HTML ஆவணங்களும் <!DOCTYPE> அறிவிப்புடன் தொடங்க வேண்டும்.

<!DOCTYPE> அறிவிப்பு ஒரு HTML குறிச்சொல் அல்ல. இது ஒரு "தகவல்" ஆகும், இது உலாவிக்கு எந்த வகையான ஆவணம் எதிர்பார்க்கப்படுகிறது என்பதை அறிவிக்கிறது.

HTML5 இல்

HTML5 இல், <!DOCTYPE> அறிவிப்பு மிகவும் எளிமையானது:

HTML5 DOCTYPE

<!DOCTYPE html>

இது HTML5 ஆவணத்தை அடையாளம் காட்டுகிறது மற்றும் அனைத்து நவீன உலாவிகளாலும் ஆதரிக்கப்படுகிறது.

பழைய ஆவணங்களில் DOCTYPE

பழைய ஆவணங்களில் (HTML 4 அல்லது XHTML), <!DOCTYPE> அறிவிப்பு மிகவும் சிக்கலானது, ஏனெனில் இது DTD (ஆவண வகை வரையறை) க்கு குறிப்பிட வேண்டும்.

HTML 4.01 Transitional DOCTYPE

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" 
"http://www.w3.org/TR/html4/loose.dtd">

XHTML 1.1 DOCTYPE

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" 
"http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

முக்கிய உதவிக்குறிப்பு:

நவீன வலைத்தளங்களுக்கு எப்போதும் HTML5 <!DOCTYPE html> ஐப் பயன்படுத்தவும். இது எளிமையானது மற்றும் அனைத்து உலாவிகளாலும் ஆதரிக்கப்படுகிறது.

இன் முக்கியத்துவம்

<!DOCTYPE> அறிவிப்பு பல காரணங்களுக்காக முக்கியமானது:

  • உலாவி இணக்க முறை: <!DOCTYPE> உலாவி எந்த இணக்க முறையில் இயங்க வேண்டும் என்பதைத் தீர்மானிக்கிறது
  • சரியான ரெண்டரிங்: இது உலாவி உங்கள் HTML குறியீட்டை சரியாக வழங்க உதவுகிறது
  • தரநிலை இணக்கம்: இது ஆவணம் தரநிலைகளைப் பின்பற்றுகிறது என்பதைக் குறிக்கிறது
  • CSS ஸ்டைலிங்: சில CSS பண்புகள் DOCTYPE சார்ந்து வேறுபடலாம்

எச்சரிக்கை:

<!DOCTYPE> இல்லாமல் உலாவி "குயிர்க் மோட்" (quirks mode) எனப்படும் பின்னோக்கு இணக்கத்தில் இயங்கும், இது பக்கத்தின் தோற்றத்தை பாதிக்கலாம்.

வெவ்வேறு DOCTYPE களில் செல்லுபடியாகும் HTML கூறுகள்

HTML5, HTML4 மற்றும் XHTML ஆகியவற்றில் பல்வேறு HTML குறிச்சொற்கள் ஆதரிக்கப்படுகின்றன:

நடைமுறை எடுத்துக்காட்டுகள்

HTML5 ஆவணம்

நவீன HTML5 ஆவணத்திற்கான எளிய தொடரியல்:

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Welcome</h1>
</body>
</html>

XHTML ஆவணம்

கடுமையான XHTML ஆவணத்திற்கான DOCTYPE:

<!DOCTYPE html PUBLIC 
"-//W3C//DTD XHTML 1.0 Strict//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

अभ्यासகள்

HTML5 ஆவணத்திற்கான சரியான <!DOCTYPE> அறிவிப்பு எது என்பதைச் சரிபார்க்கவும்:

❓ HTML5 ஆவணத்திற்கான சரியான DOCTYPE அறிவிப்பு எது?

<DOCTYPE html5>
✗ தவறு! இது செல்லுபடியாகும் DOCTYPE அல்ல
<!DOCTYPE html>
✓ சரி! இது HTML5 க்கான எளிய மற்றும் சரியான தொடரியல்
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 5.0//EN">
✗ தவறு! HTML5 இல் இது தேவையில்லை, எளிமையான வடிவம் பயன்படுத்தப்படுகிறது
<html doctype="html5">
✗ தவறு! DOCTYPE ஒரு தனி அறிவிப்பு, குறிச்சொல்லின் பண்பு அல்ல

சிறந்த நடைமுறைகள்

எப்போதும் DOCTYPE ஐச் சேர்க்கவும்

உங்கள் அனைத்து HTML ஆவணங்களிலும் முதல் வரியாக <!DOCTYPE html> ஐச் சேர்க்கவும்.

HTML5 ஐப் பயன்படுத்தவும்

புதிய திட்டங்களுக்கு எப்போதும் HTML5 DOCTYPE ஐப் பயன்படுத்தவும் - இது எளிமையானது மற்றும் நவீனமானது.

பழைய DOCTYPE களைத் தவிர்க்கவும்

புதிய திட்டங்களுக்கு HTML 4 அல்லது XHTML DOCTYPE களைப் பயன்படுத்த வேண்டாம்.

உங்கள் DOCTYPE ஐச் சரிபார்க்கவும்

உங்கள் HTML ஆவணம் செல்லுபடியாகிறதா என்பதைச் சரிபார்க்க jassif team இன் HTML உள்ளீட்டுப் பரிசோதனையைப் பயன்படுத்தவும்.
