HTML Comments

HTML टिप्पणियाँ ब्राउज़र में प्रदर्शित नहीं होती हैं, लेकिन वे आपके HTML स्रोत कोड को दस्तावेज़ीकृत करने में मदद करती हैं।

<!-- अपनी टिप्पणियाँ यहाँ लिखें -->

जसिफ टीम केयर:

प्रारंभ टैग में विस्मयादिबोधक बिंदु (!) होता है, लेकिन अंत टैग नहीं होता है।

HTML Comment Tag

आप निम्नलिखित सिंटैक्स का उपयोग करके अपने HTML स्रोत में टिप्पणियाँ जोड़ सकते हैं:

टिप्पणी वाक्यविन्यास

<!-- Write your comments here -->

जसिफ़ टीम नोट:

टिप्पणियाँ ब्राउज़र द्वारा प्रदर्शित नहीं की जाती हैं, लेकिन वे आपके HTML स्रोत कोड को दस्तावेज़ीकृत करने में मदद करती हैं।

टिप्पणी करें

टिप्पणियों के साथ आप अपने HTML कोड में सूचनाएं और अनुस्मारक रख सकते हैं:

HTML कोड

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

परिणाम (ब्राउज़र में)

This is a paragraph.

टिप्पणियाँ यहाँ प्रदर्शित नहीं की जाएंगी

सामग्री छिपाएँ

टिप्पणियों का उपयोग सामग्री को छिपाने के लिए किया जा सकता है.

सामग्री को अस्थायी रूप से छिपाना सहायक है:

HTML कोड

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph </p> -->

<p>This is a paragraph too.</p>

परिणाम (ब्राउज़र में)

This is a paragraph.

This is a paragraph too.

छिपा हुआ कॉलम यहां प्रदर्शित नहीं किया जाएगा

जसिफ़ टीम टिप:

आप एक से अधिक पंक्तियाँ छिपा सकते हैं.<!--और-->बीच में सब कुछ दृश्य से छिपा दिया जाएगा.

HTML कोड का एक टुकड़ा छिपाएँ:

<p>This is a paragraph.</p>
<!--
<p>Look at this cool image:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>This is a paragraph too.</p>

डिबगिंग के साथ HTML

HTML को डीबग करने के लिए टिप्पणियाँ भी बहुत अच्छी हैं क्योंकि आप त्रुटियों को देखने के लिए HTML कोड लाइनों पर एक समय में एक पंक्ति पर टिप्पणी कर सकते हैं।

इनलाइन सामग्री छिपाएँ

HTML कोड के मध्य में अनुभागों को छिपाने के लिए टिप्पणियों का उपयोग किया जा सकता है।

अनुच्छेद का भाग छिपाएँ:

HTML कोड

<p>This <!-- great text --> is a paragraph.</p>

परिणाम (ब्राउज़र में)

This is a paragraph.

"महान पाठ" शब्द प्रदर्शित नहीं किए जाते क्योंकि उन पर विचार किया जाता है

अभ्यास

❓HTML में टिप्पणी जोड़ने के लिए सही सिंटैक्स क्या है?

<-- This is a comment -->
✗ ग़लत! <-- एक वैध उद्घाटन टैग नहीं है
<## This is a comment ##>
✗ ग़लत! यह HTML टिप्पणी सिंटैक्स नहीं है
<--> This is a comment </-->
✗ ग़लत! HTML टिप्पणियों के लिए क्लोजिंग टैग की आवश्यकता नहीं होती है
<!-- This is a comment -->
✓ ठीक है! <!-- --> HTML टिप्पणियों के लिए सही सिंटैक्स है

जसिफ़ टीम शीर्ष युक्ति:

HTML टिप्पणियाँ हमेशा<!--प्रारंभ स्थल-->के साथ किया जाना है. विस्मयादिबोधक चिह्न (!) महत्वपूर्ण है!

टिप्पणियों के लिए सर्वोत्तम अभ्यास

महत्वपूर्ण कोड समझाइये:जटिल तर्क या काम न करने वाले कोड को समझाने के लिए टिप्पणियों का उपयोग करें।
अनुभागों को चिह्नित करें:अपनी HTML फ़ाइल के विभिन्न अनुभागों को चिह्नित करने के लिए टिप्पणियों का उपयोग करें।
अस्थायी कोड छिपाएँ:डिबगिंग या वैकल्पिक कार्यों का परीक्षण करते समय कोड छिपाएँ।
अत्यधिक टिप्पणियों से बचें:स्पष्ट कोड को टिप्पणियों की आवश्यकता नहीं है। अनावश्यक टिप्पणियाँ भ्रम पैदा कर सकती हैं।

डिबगिंग युक्ति:

डिबगिंग करते समय, त्रुटि कहां है यह जानने के लिए अपने कोड के विभिन्न हिस्सों पर एक के बाद एक टिप्पणी करें। इससे कम समय में समस्या का निदान करने में मदद मिलेगी.
