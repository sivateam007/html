HTML Text Formatting

HTML சிறப்பு அர்த்தத்துடன் உரையை வரையறுக்க பல கூறுகளைக் கொண்டுள்ளது.

எடுத்துக்காட்டு

இந்த உரை தடித்தது

இந்த உரை சாய்வு

இது H2O (கீழெழுத்து) மற்றும் x2 (மேலெழுத்து)

HTML Formatting Elements

வடிவமைப்பு கூறுகள் சிறப்பு வகையான உரையைக் காட்டுவதற்காக வடிவமைக்கப்பட்டன:

<b> - தடித்த உரை

முக்கியத்துவம் இல்லாமல் தடித்த உரை

<strong> - முக்கியமான உரை

வலுவான முக்கியத்துவம் கொண்ட உரை

<i> - சாய்வு உரை

மாற்று குரல் அல்லது மனநிலையில் உரை

<em> - வலியுறுத்தப்பட்ட உரை

வலியுறுத்தப்படும் உரை

HTML <b> மற்றும் <strong> கூறுகள்

HTML <b> கூறு தடித்த உரையை வரையறுக்கிறது, எந்த கூடுதல் முக்கியத்துவமும் இல்லாமல்.

எடுத்துக்காட்டு: <b>

<b>This text is bold</b>

இந்த உரை தடித்தது

எடுத்துக்காட்டு: <strong>

<strong>This text is important!</strong>

இந்த உரை முக்கியமானது!

Jassif Team உதவிக்குறிப்பு:

<strong> கூறு உள்ளடக்கத்தின் முக்கியத்துவத்தை குறிக்கிறது, அதே வேளையில் <b> கூறு காட்சி தோற்றத்தை மட்டுமே குறிக்கிறது.

HTML <i> மற்றும் <em> கூறுகள்

HTML <i> கூறு மாற்று குரல் அல்லது மனநிலையில் உரையின் ஒரு பகுதியை வரையறுக்கிறது. सामग्री பொதுவாக சாய்வில் காட்டப்படும்.

Jassif Team உதவிக்குறிப்பு:

<i> குறிச்சொல் பெரும்பாலும் ஒரு தொழில்நுட்ப சொல், மற்றொரு மொழியில் இருந்து ஒரு சொற்றொடர், ஒரு சிந்தனை, கப்பல் பெயர் போன்றவற்றைக் குறிக்க பயன்படுகிறது.

எடுத்துக்காட்டு: <i>

<i>This text is italic</i>

இந்த உரை சாய்வு

எடுத்துக்காட்டு: <em>

<em>This text is emphasized</em>

இந்த உரை வலியுறுத்தப்பட்டது

குறிப்பு:

ஒரு திரை வாசிப்பான் <em> இல் உள்ள வார்த்தைகளை வாய்மொழி மன அழுத்தத்தைப் பயன்படுத்தி, அழுத்தத்துடன் உச்சரிக்கும்.

மற்ற HTML வடிவமைப்பு கூறுகள்

<small> கூறு

சிறிய உரையை வரையறுக்கிறது:

<small>This is some smaller text.</small>

இது சாதாரண உரை இது சிறிய உரை

<mark> கூறு

குறிக்கப்பட வேண்டிய அல்லது சிறப்பாகக் காட்டப்பட வேண்டிய உரையை வரையறுக்கிறது:

<p>Do not forget to buy <mark>milk</mark> today.</p>

இன்று பால் வாங்க மறக்காதீர்கள்.

<del> கூறு

ஒரு ஆவணத்திலிருந்து நீக்கப்பட்ட உரையை வரையறுக்கிறது:

<p>My favorite color is <del>blue</del> red.</p>

எனக்குப் பிடித்த நிறம் நீலம் சிவப்பு.

<ins> கூறு

ஒரு ஆவணத்தில் செருகப்பட்ட உரையை வரையறுக்கிறது:

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

எனக்குப் பிடித்த நிறம் நீலம் சிவப்பு.

<sub> கூறு

கீழெழுத்து உரையை வரையறுக்கிறது. கீழெழுத்து உரை சாதாரண வரியில் பாதி எழுத்துக்குக் கீழே தோன்றும்:

<p>This is <sub>subscripted</sub> text.</p>

இது H2O (நீர்) - ரசாயன சூத்திரத்திற்கு பயன்படுகிறது

<sup> கூறு

மேலெழுத்து உரையை வரையறுக்கிறது. மேலெழுத்து உரை சாதாரண வரியில் பாதி எழுத்துக்கு மேல் தோன்றும்:

<p>This is <sup>superscripted</sup> text.</p>

இது WWW[1] - அடிக்குறிப்புகளுக்கு பயன்படுகிறது

अभ्यास

❓ பின்வரும் HTML கூறுகளில் எது உரையை தடித்ததாக ஆக்குகிறது?

<em> மற்றும் <b>
✗ தவறு! <em> உரையை சாய்வாக மாற்றுகிறது, தடித்ததாக அல்ல
<strong> மற்றும் <b>
✓ சரி! <strong> மற்றும் <b> இரண்டும் உரையை தடித்ததாக காட்டுகின்றன
<big> மற்றும் <b>
✗ தவறு! <big> என்பது HTML5 இல் ஆதரவற்றது
<mark> மற்றும் <b>
✗ தவறு! <mark> உரையை சிறப்பாக காட்டுகிறது, தடித்ததாக அல்ல

Jassif Team முக்கிய உதவிக்குறிப்பு:

<strong> மற்றும் <b> இரண்டும் காட்சிப்படுத்தப்படும் போது தடித்த உரையை உருவாக்குகின்றன, ஆனால் <strong> சொற்பொருள் முக்கியத்துவத்தையும் கொண்டுள்ளது.

சிறந்த நடைமுறைகள்

பொருத்தமான கூறுகளைப் பயன்படுத்தவும்: தோற்றத்திற்காக மட்டும் <b> பயன்படுத்தவும், உள்ளடக்க முக்கியத்துவத்திற்கு <strong> பயன்படுத்தவும்.
செமான்டிக்ஸைப் பராமரிக்கவும்: சாய்வு தோற்றத்திற்காக மட்டும் <i> பயன்படுத்தவும், உச்சரிப்புக்கு <em> பயன்படுத்தவும்.
CSS ஐ முன்னுரிமை அளிக்கவும்: தூய தோற்ற வடிவமைப்பிற்கு CSS பயன்படுத்தவும், HTML வடிவமைப்பு கூறுகள் அல்ல.
காலாவதியான கூறுகளைத் தவிர்க்கவும்: <font>, <center>, <big> போன்ற காலாவதியான கூறுகளைப் பயன்படுத்த வேண்டாம்.
