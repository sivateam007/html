HTML பக்கத் தலைப்பு

ஒவ்வொரு வலைப்பக்கமும் பக்கத்தின் பொருளை விவரிக்கும் ஒரு பக்கத் தலைப்பைக் கொண்டிருக்க வேண்டும்.

தலைப்பு உறுப்பு

<title> உறுப்பு உங்கள் பக்கத்தில் ஒரு தலைப்பைச் சேர்க்கிறது:

எடுத்துக்காட்டு

<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
</head>
<body>

The content of the document......

</body>
</html>

தலைப்பு உலாவியின் தலைப்புப் பட்டியில் காட்டப்படுகிறது:

ஒரு நல்ல தலைப்பு என்றால் என்ன?

தலைப்பு सामग्री மற்றும் பக்கத்தின் பொருளை விவரிக்க வேண்டும்.

பக்கத் தலைப்பு தேடுபொறி உகப்பாக்கத்திற்கு (SEO) மிகவும் முக்கியமானது. தேடல் முடிவுகளில் பக்கங்களைப் பட்டியலிடும் போது வரிசையைத் தீர்மானிக்க தேடுபொறி வழிமுறைகளால் உரை பயன்படுத்தப்படுகிறது.

<title> உறுப்பு:

  • உலாவி கருவிப்பட்டியில் ஒரு தலைப்பை வரையறுக்கிறது
  • பக்கம் பிடித்தவைகளில் சேர்க்கப்படும் போது பக்கத்திற்கு ஒரு தலைப்பை வழங்குகிறது
  • தேடுபொறி முடிவுகளில் பக்கத்திற்கான தலைப்பைக் காட்டுகிறது

எனவே, தலைப்பை முடிந்தவரை துல்லியமாகவும் அர்த்தமுள்ளதாகவும் உருவாக்க முயற்சிக்கவும்!

HTML தலைப்பு குறிச்சொல்

குறிச்சொல் விளக்கம்
<title> ஆவணத்தின் தலைப்பை வரையறுக்கிறது

அனைத்து கிடைக்கக்கூடிய HTML குறிச்சொற்களின் முழுமையான பட்டியலுக்கு, எங்கள் HTML குறிச்சொல் குறிப்பைப் பார்வையிடவும்.
