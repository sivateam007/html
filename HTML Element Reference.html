HTML Element Reference

HTML குறிச்சொற்கள் அகரவரிசைப்படி

குறிச்சொல் விளக்கம்
<!--...--> கருத்தை வரையறுக்கிறது
<!DOCTYPE> ஆவண வகையை வரையறுக்கிறது
<a> ஹைப்பர்லிங்கை வரையறுக்கிறது
<abbr> சுருக்கத்தை அல்லது சுருக்கெழுத்தை வரையறுக்கிறது
<acronym> HTML5 இல் ஆதரிக்கப்படவில்லை. <abbr> பயன்படுத்தவும்.
சுருக்கெழுத்தை வரையறுக்கிறது
<address> ஆவணத்தின் ஆசிரியர்/உரிமையாளரின் தொடர்புத் தகவலை வரையறுக்கிறது
<applet> HTML5 இல் ஆதரிக்கப்படவில்லை. <embed> அல்லது <object> பயன்படுத்தவும்.
உள்ளமைக்கப்பட்ட ஆப்லெட்டை வரையறுக்கிறது
<area> பட வரைபடத்தின் உள்ளே உள்ள பகுதியை வரையறுக்கிறது
<article> கட்டுரையை வரையறுக்கிறது
<aside> பக்க உள்ளடக்கத்திலிருந்து விலக்கான உள்ளடக்கத்தை வரையறுக்கிறது
<audio> உள்ளமைக்கப்பட்ட ஒலி உள்ளடக்கத்தை வரையறுக்கிறது
<b> தடித்த உரையை வரையறுக்கிறது
<base> ஒரு ஆவணத்தில் உள்ள அனைத்து சார்பு URL களுக்கான அடிப்படை URL/இலக்கைக் குறிப்பிடுகிறது
<basefont> HTML5 இல் ஆதரிக்கப்படவில்லை. CSS பயன்படுத்தவும்.
ஒரு ஆவணத்தில் உள்ள அனைத்து உரைக்கும் இயல்புநிலை நிறம், அளவு மற்றும் எழுத்துருவைக் குறிப்பிடுகிறது
<bdi> வெளியே உள்ள பிற உரையிலிருந்து வேறுபட்ட திசையில் வடிவமைக்கப்படக்கூடிய உரையின் ஒரு பகுதியைத் தனிமைப்படுத்துகிறது
<bdo> தற்போதைய உரை திசையை மேலெழுதுகிறது
<big> HTML5 இல் ஆதரிக்கப்படவில்லை. CSS பயன்படுத்தவும்.
பெரிய உரையை வரையறுக்கிறது
<blockquote> மற்றொரு மூலத்திலிருந்து மேற்கோள் காட்டப்பட்ட பகுதியை வரையறுக்கிறது
<body> ஆவணத்தின் உடலை வரையறுக்கிறது
<br> ஒற்றை வரி முறிப்பை வரையறுக்கிறது
<button> கிளிக் செய்யக்கூடிய பொத்தானை வரையறுக்கிறது
<canvas> வரைகலைகளை வரைய பயன்படுகிறது, ஸ்கிரிப்டிங் மூலம் (பொதுவாக JavaScript)
<caption> அட்டவணை தலைப்பை வரையறுக்கிறது
<center> HTML5 இல் ஆதரிக்கப்படவில்லை. CSS பயன்படுத்தவும்.
மையப்படுத்தப்பட்ட உரையை வரையறுக்கிறது
<cite> ஒரு படைப்பின் தலைப்பை வரையறுக்கிறது
<code> கணினி குறியீட்டின் துண்டை வரையறுக்கிறது
<col> ஒரு <colgroup> உறுப்புக்குள் உள்ள ஒவ்வொரு நெடுவரிசைக்கும் பண்புகளைக் குறிப்பிடுகிறது
<colgroup> வடிவமைப்பிற்கான அட்டவணையில் உள்ள ஒன்று அல்லது அதற்கு மேற்பட்ட நெடுவரிசைகளின் குழுவைக் குறிப்பிடுகிறது
<data> கொடுக்கப்பட்ட உள்ளடக்கத்தின் இயந்திரம்-படிக்கக்கூடிய மொழிபெயர்ப்பைச் சேர்க்கிறது
<datalist> உள்ளீட்டு கட்டுப்பாடுகளுக்கான முன்-வரையறுக்கப்பட்ட விருப்பங்களின் பட்டியலைக் குறிப்பிடுகிறது
<dd> விளக்கப் பட்டியலில் உள்ள ஒரு சொல்லின் விளக்கத்தை/மதிப்பை வரையறுக்கிறது
<del> ஆவணத்திலிருந்து நீக்கப்பட்ட உரையை வரையறுக்கிறது
<details> பயனர் காணலாம் அல்லது மறைக்கலாம் என்ற கூடுதல் விவரங்களை வரையறுக்கிறது
<dfn> உள்ளடக்கத்திற்குள் வரையறுக்கப்படும் ஒரு சொல்லைக் குறிப்பிடுகிறது
<dialog> உரையாடல் பெட்டி அல்லது சாளரத்தை வரையறுக்கிறது
<dir> HTML5 இல் ஆதரிக்கப்படவில்லை. <ul> பயன்படுத்தவும்.
அடைவு பட்டியலை வரையறுக்கிறது
<div> ஆவணத்தில் உள்ள ஒரு பகுதியை வரையறுக்கிறது
<dl> விளக்கப் பட்டியலை வரையறுக்கிறது
<dt> விளக்கப் பட்டியலில் உள்ள ஒரு சொல்லை/பெயரை வரையறுக்கிறது
<em> வலியுறுத்தப்பட்ட உரையை வரையறுக்கிறது
<embed> வெளி பயன்பாட்டிற்கான கொள்கலனை வரையறுக்கிறது
<fieldset> படிவத்தில் தொடர்புடைய உறுப்புகளைக் குழுவாக்குகிறது
<figcaption> <figure> உறுப்புக்கான தலைப்பை வரையறுக்கிறது
<figure> சுய-உள்ளடக்க உள்ளடக்கத்தைக் குறிப்பிடுகிறது
<font> HTML5 இல் ஆதரிக்கப்படவில்லை. CSS பயன்படுத்தவும்.
உரைக்கான எழுத்துரு, நிறம் மற்றும் அளவை வரையறுக்கிறது
<footer> ஆவணம் அல்லது பகுதிக்கான அடிக்குறிப்பை வரையறுக்கிறது
<form> பயனர் உள்ளீட்டிற்கான HTML படிவத்தை வரையறுக்கிறது
<frame> HTML5 இல் ஆதரிக்கப்படவில்லை.
பிரேம்செட்டில் உள்ள சாளரத்தை (பிரேம்) வரையறுக்கிறது
<frameset> HTML5 இல் ஆதரிக்கப்படவில்லை.
பிரேம்களின் தொகுப்பை வரையறுக்கிறது
<h1> to <h6> HTML தலைப்புகளை வரையறுக்கிறது
<head> ஆவணத்திற்கான மெட்டாடேட்டா/தகவல்களைக் கொண்டுள்ளது
<header> ஆவணம் அல்லது பகுதிக்கான தலைப்பை வரையறுக்கிறது
<hgroup> தலைப்பு மற்றும் தொடர்புடைய உள்ளடக்கத்தை வரையறுக்கிறது
<hr> உள்ளடக்கத்தில் உள்ள கருப்பொருள் மாற்றத்தை வரையறுக்கிறது
<html> HTML ஆவணத்தின் மூலத்தை வரையறுக்கிறது
<i> மாற்று குரல் அல்லது மனநிலையில் உள்ள உரையின் ஒரு பகுதியை வரையறுக்கிறது
<iframe> இன்லைன் பிரேமை வரையறுக்கிறது
<img> படத்தை வரையறுக்கிறது
<input> உள்ளீட்டு கட்டுப்பாட்டை வரையறுக்கிறது
<ins> ஆவணத்தில் சேர்க்கப்பட்ட உரையை வரையறுக்கிறது
<kbd> விசைப்பலகை உள்ளீட்டை வரையறுக்கிறது
<label> <input> உறுப்புக்கான லேபிளை வரையறுக்கிறது
<legend> <fieldset> உறுப்புக்கான தலைப்பை வரையறுக்கிறது
<li> பட்டியல் உருப்படியை வரையறுக்கிறது
<link> ஆவணத்திற்கும் வெளி வளத்திற்கும் இடையிலான உறவை வரையறுக்கிறது (பெரும்பாலும் ஸ்டைல் ஷீட்டுகளுடன் இணைக்கப் பயன்படுகிறது)
<main> ஆவணத்தின் முக்கிய உள்ளடக்கத்தைக் குறிப்பிடுகிறது
<map> பட வரைபடத்தை வரையறுக்கிறது
<mark> குறிக்கப்பட்ட/முன்னிலைப்படுத்தப்பட்ட உரையை வரையறுக்கிறது
<menu> வரிசையிடப்படாத பட்டியலை வரையறுக்கிறது
<meta> HTML ஆவணத்தைப் பற்றிய மெட்டாடேட்டாவை வரையறுக்கிறது
<meter> அறியப்பட்ட வரம்பிற்குள் உள்ள அளவிடல் அளவீட்டை வரையறுக்கிறது (அளவி)
<nav> வழிசெலுத்தல் இணைப்புகளை வரையறுக்கிறது
<noframes> HTML5 இல் ஆதரிக்கப்படவில்லை.
பிரேம்களை ஆதரிக்காத பயனர்களுக்கான மாற்று உள்ளடக்கத்தை வரையறுக்கிறது
<noscript> கிளையன்-சைடு ஸ்கிரிப்ட்களை ஆதரிக்காத பயனர்களுக்கான மாற்று உள்ளடக்கத்தை வரையறுக்கிறது
<object> வெளி பயன்பாட்டிற்கான கொள்கலனை வரையறுக்கிறது
<ol> வரிசைப்படுத்தப்பட்ட பட்டியலை வரையறுக்கிறது
<optgroup> டிராப்பவுன் பட்டியலில் உள்ள தொடர்புடைய விருப்பங்களின் குழுவை வரையறுக்கிறது
<option> டிராப்பவுன் பட்டியலில் உள்ள விருப்பத்தை வரையறுக்கிறது
<output> கணக்கீட்டின் முடிவை வரையறுக்கிறது
<p> பத்தியை வரையறுக்கிறது
<param> பொருளுக்கான அளவுருவை வரையறுக்கிறது
<picture> பல பட வளங்களுக்கான கொள்கலனை வரையறுக்கிறது
<pre> முன்வடிவமைக்கப்பட்ட உரையை வரையறுக்கிறது
<progress> பணியின் முன்னேற்றத்தைக் குறிப்பிடுகிறது
<q> குறுகிய மேற்கோளை வரையறுக்கிறது
<rp> ரூபி குறிப்புகளை ஆதரிக்காத உலாவிகளில் காட்டுவதை வரையறுக்கிறது
<rt> எழுத்துக்களின் விளக்கம்/உச்சரிப்பை வரையறுக்கிறது (கிழக்காசிய அச்சுக்கலையுக்கு)
<ruby> ரூபி குறிப்பை வரையறுக்கிறது (கிழக்காசிய அச்சுக்கலையுக்கு)
<s> இனி சரியானதல்ல என்ற உரையை வரையறுக்கிறது
<samp> கணினி நிரலின் மாதிரி வெளியீட்டை வரையறுக்கிறது
<script> கிளையன்-சைடு ஸ்கிரிப்ட்டை வரையறுக்கிறது
<search> தேடல் பகுதியை வரையறுக்கிறது
