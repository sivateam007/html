File Path Examples

Path Description

<img src="images/picture.jpg"> "चित्र.jpg" फ़ाइल वर्तमान फ़ोल्डर में छवि फ़ोल्डर में स्थित है

HTML File Paths

फ़ाइल पथ किसी वेबसाइट की फ़ोल्डर संरचना में फ़ाइल के स्थान का वर्णन करता है।

फ़ाइल पथ का उपयोग बाहरी फ़ाइलों से लिंक करते समय किया जाता है, उदाहरण के लिए:

  • वेब पेज
  • चित्र
  • स्टाइल शीट
  • JavaScripts

Absolute File Paths

एक संपूर्ण फ़ाइल पथ किसी फ़ाइल का पूर्ण URL होता है:

Example

<img src="https://www.jassifteam.com/images/picture.jpg" alt="Mountain">

The <img> tag is explained in the chapter: HTML Images.

Relative File Paths

एक सापेक्ष फ़ाइल पथ वर्तमान पृष्ठ से संबंधित फ़ाइल को इंगित करता है।

निम्नलिखित उदाहरण में, फ़ाइल पथ वर्तमान वेबसाइट के मूल में स्थित छवि फ़ोल्डर में एक फ़ाइल को इंगित करता है:

Example

<img src="/images/picture.jpg" alt="Mountain">

निम्नलिखित उदाहरण में, फ़ाइल पथ वर्तमान फ़ोल्डर में स्थित छवि फ़ोल्डर में एक फ़ाइल को इंगित करता है:

Example

<img src="images/picture.jpg" alt="Mountain">

निम्नलिखित उदाहरण में, फ़ाइल पथ वर्तमान फ़ोल्डर से एक स्तर ऊपर छवि फ़ोल्डर में एक फ़ाइल को इंगित करता है:

Example

<img src="../images/picture.jpg" alt="Mountain">

Best Practice

सबसे अच्छा अभ्यास सापेक्ष फ़ाइल पथ (यदि संभव हो) का उपयोग करना है।

सापेक्ष फ़ाइल पथों का उपयोग करते समय, आपके वेब पेज आपके वर्तमान आधार URL से बंधे नहीं होंगे। सभी लिंक आपके अपने कंप्यूटर (लोकलहोस्ट) और आपके वर्तमान सार्वजनिक डोमेन और आपके भविष्य के सार्वजनिक डोमेन पर काम करते हैं।
