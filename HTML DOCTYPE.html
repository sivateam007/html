HTML DOCTYPE

HTML ஆவண வகை அறிவிப்பு

HTML ஆவண வகை

அனைத்து HTML ஆவணங்களும் DOCTYPE அறிவிப்புடன் தொடங்க வேண்டும்.

அறிவிப்பு ஒரு HTML குறிச்சொல் அல்ல. இது உலாவிக்கு எந்த ஆவண வகை எதிர்பார்க்கப்படுகிறது என்பதை அறிவிக்கும் "தகவல்" ஆகும்.

HTML5 இல், DOCTYPE அறிவிப்பு எளிமையானது:

<!DOCTYPE html>

பழைய ஆவணங்களில் (HTML 4 அல்லது XHTML), அறிவிப்பு மிகவும் சிக்கலானது, ஏனெனில் அறிவிப்பு DTD (ஆவண வகை வரையறை) க்கு குறிப்பிட வேண்டும்.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

ஆவண வகைகளைப் பற்றி மேலும் அறிய DOCTYPE குறிப்பில் jassif team இல் படிக்கலாம்.

अभ्यासகள்

❓ HTML5 இல் சரியான DOCTYPE அறிவிப்பு எது?

<DOCTYPE html5>
✗ தவறு! இது செல்லுபடியாகும் DOCTYPE அல்ல
<!DOCTYPE html>
✓ சரி! இது HTML5 க்கான எளிய மற்றும் சரியான தொடரியல்
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 5.0//EN">
✗ தவறு! HTML5 இல் இது தேவையில்லை, எளிமையான வடிவம் பயன்படுத்தப்படுகிறது
