HTML Buttons

बटन उपयोगकर्ताओं को वेब पेज के साथ इंटरैक्ट करने की अनुमति देते हैं। वे फ़ॉर्म सबमिट कर सकते हैं, जावास्क्रिप्ट निष्पादित कर सकते हैं, या क्लिक करने पर विभिन्न क्रियाएं ट्रिगर कर सकते हैं।

HTML Button

HTML <बटन> तत्व एक क्लिक करने योग्य बटन को परिभाषित करता है।

अपने आप में, एक बटन तब तक कुछ नहीं करता जब तक आप उसमें कोई क्रिया नहीं जोड़ते।

Example

<button>Click Me</button>

Styling HTML Buttons

बटनों को अक्सर CSS के साथ स्टाइल किया जाता है:

Example

<button class="mytestbtn">Green Button</button>

Disabled Buttons

किसी बटन को क्लिक न करने योग्य बनाने के लिए अक्षम विशेषता का उपयोग करें:

Example

<button disabled>Disabled Button</button>

बख्शीश:

अक्षम बटन क्लिक करने योग्य नहीं होते हैं और आमतौर पर धुंधले दिखाई देते हैं।

Button with JavaScript

जब उपयोगकर्ता ऑनक्लिक विशेषता का उपयोग करके किसी बटन पर क्लिक करता है तो आप जावास्क्रिप्ट निष्पादित कर सकते हैं:

Example

<button onclick="alert('Hello!')">Click Me</button>

टिप्पणी:

आप हमारे HTML जावास्क्रिप्ट अध्याय में जावास्क्रिप्ट के बारे में अधिक जानेंगे।

Button Types

प्रकार विशेषता परिभाषित करती है कि क्लिक करने पर बटन क्या करता है। तीन प्रकार के बटन हैं:

type="button" - A normal clickable button (does nothing by default)
type="submit" - Submits a form
type="reset" - Resets all form fields
<button type="button">Normal Button</button>
<button type="submit">Submit</button>
<button type="reset">Reset</button>

बटनों का उपयोग अक्सर प्रपत्रों के अंदर किया जाता है, जिसके बारे में आप बाद के अध्याय में जानेंगे।

अभी के लिए, ध्यान दें कि सबमिट बटन सर्वर पर फॉर्म डेटा भेजता है, जबकि रीसेट बटन फॉर्म को साफ़ करता है:

Example

<form action="/action_page.php">
  First name: <input type="text" name="fname">
  <button type="submit">Submit</button>
  <button type="reset">Reset Form</button>
</form>

टिप्पणी:

आपको हमेशा प्रकार विशेषता निर्दिष्ट करनी होगी। एक फॉर्म के भीतर, डिफ़ॉल्ट प्रकार सबमिट होता है, और यदि प्रकार छोड़ दिया जाता है तो ब्राउज़र अलग-अलग व्यवहार कर सकते हैं।
