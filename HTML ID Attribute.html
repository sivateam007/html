HTML id Attribute

HTML आईडी विशेषता का उपयोग HTML तत्व के लिए एक अद्वितीय आईडी निर्दिष्ट करने के लिए किया जाता है।

एक HTML दस्तावेज़ में एक ही आईडी वाले एक से अधिक तत्व नहीं हो सकते।

The id Attribute

आईडी विशेषता HTML तत्व के लिए एक अद्वितीय आईडी निर्दिष्ट करती है। आईडी विशेषता का मान HTML दस्तावेज़ में अद्वितीय होना चाहिए।

आईडी विशेषता का उपयोग स्टाइल शीट में एक विशिष्ट शैली घोषणा को इंगित करने के लिए किया जाता है। निर्दिष्ट आईडी के साथ तत्व तक पहुंचने और हेरफेर करने के लिए जावास्क्रिप्ट द्वारा भी उपयोग किया जाता है।

आईडी के लिए सिंटैक्स: एक हैश कैरेक्टर (#) के बाद एक आईडी नाम लिखें। फिर, घुंघराले ब्रेसिज़ { } के अंदर सीएसएस विशेषताओं को परिभाषित करें।

Example

निम्नलिखित उदाहरण में एक <h1> तत्व है जो आईडी नाम "myHeader" की ओर इंगित करता है। इस <h1> तत्व को हेड सेक्शन में #myHeader शैली परिभाषा के अनुसार स्टाइल किया जाएगा:

<!DOCTYPE html>
<html>
<head>
<style>
#myHeader {
  background-color: lightblue;
  color: black;
  padding: 40px;
  text-align: center;
}
</style>
</head>
<body>

<h1 id="myHeader">My Header</h1>

</body>
</html>

टिप्पणी:

आईडी नाम केस संवेदी है!

टिप्पणी:

आईडी नाम में कम से कम एक अक्षर होना चाहिए, किसी संख्या से शुरू नहीं होना चाहिए और इसमें सफेद रिक्त स्थान (रिक्त स्थान, टैब आदि) नहीं होना चाहिए।

Difference Between Class and ID

एक वर्ग नाम का उपयोग कई HTML तत्वों द्वारा किया जा सकता है, लेकिन एक आईडी नाम का उपयोग पृष्ठ पर केवल एक HTML तत्व द्वारा किया जाना चाहिए:

<style>
/* Style the element with the id "myHeader" */
#myHeader {
  background-color: lightblue;
  color: black;
  padding: 40px;
  text-align: center;
}

/* Style all elements with the class name "city" */
.city {
  background-color: tomato;
  color: white;
  padding: 10px;
}
</style>

<!-- An element with a unique id -->
<h1 id="myHeader">My Cities</h1>

<!-- Multiple elements with same class -->
<h2 class="city">London</h2>
<p>London is the capital of England.</p>

<h2 class="city">Paris</h2>
<p>Paris is the capital of France.</p>

<h2 class="city">Tokyo</h2>
<p>Tokyo is the capital of Japan.</p>

बख्शीश:

हमारे सीएसएस ट्यूटोरियल से सीएसएस के बारे में और जानें।

HTML Bookmarks with ID and Links

HTML बुकमार्क का उपयोग पाठकों को वेब पेज के विशिष्ट क्षेत्रों में नेविगेट करने की अनुमति देने के लिए किया जाता है।

यदि आपका पृष्ठ बहुत लंबा है तो बुकमार्क उपयोगी होते हैं।

बुकमार्क का उपयोग करने के लिए, आपको पहले इसे बनाना होगा और फिर इसमें एक लिंक जोड़ना होगा।

फिर, जब लिंक पर क्लिक किया जाता है, तो पृष्ठ स्क्रॉल होकर वहां पहुंच जाता है जहां बुकमार्क है।

Example

First, create a bookmark with the id attribute:

<h2 id="C4">Chapter 4</h2>

Then, add a link to the bookmark ("Jump to Chapter 4"), from within the same page:

Example

<a href="#C4">Jump to Chapter 4</a>

Or, add a link to the bookmark ("Jump to Chapter 4"), from another page:

<a href="html_demo.html#C4">Jump to Chapter 4</a>

Using the id Attribute in JavaScript

आईडी विशेषता का उपयोग जावास्क्रिप्ट द्वारा उस विशेष तत्व के लिए कुछ कार्य करने के लिए भी किया जा सकता है।

आप जावास्क्रिप्ट getElementById() विधि से एक विशिष्ट आईडी वाले तत्व तक पहुंच सकते हैं:

Example

Use the id attribute to manipulate text with JavaScript:

<script>
function displayResult() {
  document.getElementById("myHeader").innerHTML = "Have a nice day!";
}
</script>

बख्शीश:

HTML जावास्क्रिप्ट अध्याय में जावास्क्रिप्ट पढ़ें, या हमारा जावास्क्रिप्ट ट्यूटोरियल पढ़ें।
