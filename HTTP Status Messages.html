HTTP நிலை செய்திகள்

HTML பிழை செய்திகள்

ஒரு உலாவி ஒரு வலை சேவையகத்திடமிருந்து சேவையைக் கோரும் போது, ஒரு பிழை ஏற்படலாம், மேலும் சேவையகம் "404 Not Found" போன்ற ஒரு பிழைக் குறியீட்டைத் திருப்பித் தரலாம்.

இந்த பிழைகளை HTML பிழை செய்திகள் என்று அழைப்பது பொதுவானது.

ஆனால் இந்த செய்திகள் HTTP நிலை செய்திகள் என்று அழைக்கப்படுகின்றன. உண்மையில், சேவையகம் ஒவ்வொரு கோரிக்கைக்கும் எப்போதும் ஒரு செய்தியைத் திருப்பித் தருகிறது. மிகவும் பொதுவான செய்தி 200 OK ஆகும்.

கீழே திருப்பித் தரப்படக்கூடிய HTTP நிலை செய்திகளின் பட்டியல் உள்ளது:

1xx: தகவல்

செய்தி விளக்கம்
100 Continue சேவையகம் கோரிக்கை தலைப்புகளைப் பெற்றுள்ளது, மேலும் வாடிக்கையாளர் கோரிக்கை உடலை அனுப்பத் தொடங்க வேண்டும்
101 Switching Protocols கோரிக்கையாளர் சேவையகத்தை நெறிமுறைகளை மாற்றுமாறு கோரியுள்ளார்
103 Early Hints சேவையகம் பதிலைத் தயாரிக்கும்போது உலாவி வளங்களை முன்கூட்டியே ஏற்றத் தொடங்க அனுமதிக்க Link தலைப்புடன் பயன்படுத்தப்படுகிறது

2xx: வெற்றிகரமானது

செய்தி விளக்கம்
200 OK கோரிக்கை சரியானது (இது வெற்றிகரமான HTTP கோரிக்கைகளுக்கான நிலையான பதில்)
201 Created கோரிக்கை நிறைவேற்றப்பட்டது, மேலும் ஒரு புதிய வளம் உருவாக்கப்பட்டது
202 Accepted கோரிக்கை செயலாக்கத்திற்காக ஏற்கப்பட்டுள்ளது, ஆனால் செயலாக்கம் முடிவடையவில்லை
203 Non-Authoritative Information கோரிக்கை வெற்றிகரமாக செயலாக்கப்பட்டது, ஆனால் வேறொரு மூலத்திலிருந்து வரக்கூடிய தகவல்களைத் திருப்பித் தருகிறது
204 No Content கோரிக்கை வெற்றிகரமாக செயலாக்கப்பட்டது, ஆனால் எந்த உள்ளடக்கத்தையும் திருப்பித் தரவில்லை
205 Reset Content கோரிக்கை வெற்றிகரமாக செயலாக்கப்பட்டது, ஆனால் எந்த உள்ளடக்கத்தையும் திருப்பித் தரவில்லை, மேலும் கோரிக்கையாளர் ஆவணக் காட்சியை மீட்டமைக்க வேண்டும்
206 Partial Content வாடிக்கையாளரால் அனுப்பப்பட்ட range தலைப்பு காரணமாக சேவையகம் வளத்தின் ஒரு பகுதியை மட்டுமே வழங்குகிறது

3xx: திருப்பிவிடுதல்

செய்தி விளக்கம்
300 Multiple Choices ஒரு இணைப்புப் பட்டியல். பயனர் ஒரு இணைப்பைத் தேர்ந்தெடுத்து அந்த இடத்திற்குச் செல்லலாம். அதிகபட்சம் ஐந்து முகவரிகள்
301 Moved Permanently கோரப்பட்ட பக்கம் ஒரு புதிய URL க்கு நிரந்தரமாக நகர்த்தப்பட்டது
302 Found கோரப்பட்ட பக்கம் தற்காலிகமாக ஒரு புதிய URL க்கு நகர்த்தப்பட்டது
303 See Other கோரப்பட்ட பக்கம் வேறொரு URL க்கு கீழ் காணலாம்
304 Not Modified கோரப்பட்ட பக்கம் கடைசியாகக் கோரப்பட்டதிலிருந்து மாற்றப்படவில்லை என்பதைக் குறிக்கிறது
307 Temporary Redirect கோரப்பட்ட பக்கம் தற்காலிகமாக ஒரு புதிய URL க்கு நகர்த்தப்பட்டது
308 Permanent Redirect கோரப்பட்ட பக்கம் நிரந்தரமாக ஒரு புதிய URL க்கு நகர்த்தப்பட்டது

4xx: வாடிக்கையாளர் பிழை

செய்தி விளக்கம்
400 Bad Request மோசமான தொடரியல் காரணமாக கோரிக்கையை நிறைவேற்ற முடியாது
401 Unauthorized கோரிக்கை சட்டப்பூர்வ கோரிக்கையாக இருந்தது, ஆனால் சேவையகம் அதற்குப் பதிலளிக்க மறுக்கிறது. அங்கீகாரம் சாத்தியமாக இருக்கும் போது பயன்படுத்த, ஆனால் தோல்வியடைந்தது அல்லது இன்னும் வழங்கப்படவில்லை
402 Payment Required எதிர்கால பயன்பாட்டிற்காக ஒதுக்கப்பட்டது
403 Forbidden கோரிக்கை சட்டப்பூர்வ கோரிக்கையாக இருந்தது, ஆனால் சேவையகம் அதற்குப் பதிலளிக்க மறுக்கிறது
404 Not Found கோரப்பட்ட பக்கத்தைக் கண்டறிய முடியவில்லை, ஆனால் எதிர்காலத்தில் மீண்டும் கிடைக்கலாம்
405 Method Not Allowed ஒரு பக்கத்தின் கோரிக்கை அந்தப் பக்கத்தால் ஆதரிக்கப்படாத கோரிக்கை முறையைப் பயன்படுத்தி செய்யப்பட்டது
406 Not Acceptable சேவையகம் வாடிக்கையாளரால் ஏற்றுக்கொள்ளப்படாத பதிலை மட்டுமே உருவாக்க முடியும்
407 Proxy Authentication Required வாடிக்கையாளர் முதலில் ப்ராக்ஸியுடன் தன்னை அங்கீகரிக்க வேண்டும்
408 Request Timeout கோரிக்கைக்காக காத்திருக்கும் போது சேவையகத்தின் நேரம் முடிந்துவிட்டது
409 Conflict கோரிக்கையில் ஒரு மோதல் இருப்பதால் கோரிக்கையை முடிக்க முடியவில்லை
410 Gone கோரப்பட்ட பக்கம் இனி கிடைக்கவில்லை
411 Length Required "Content-Length" வரையறுக்கப்படவில்லை. சேவையகம் அதை இல்லாமல் கோரிக்கையை ஏற்காது
412 Precondition Failed கோரிக்கையில் கொடுக்கப்பட்ட முன்நிபந்தனை சேவையகத்தால் தவறானதாக மதிப்பிடப்பட்டது
413 Request Too Large கோரிக்கை நிறுவனம் மிகவும் பெரியது என்பதால் சேவையகம் கோரிக்கையை ஏற்காது
414 Request-URI Too Long URI மிக நீளமாக இருப்பதால் சேவையகம் கோரிக்கையை ஏற்காது. நீண்ட வினவல் தகவலுடன் POST கோரிக்கையை GET கோரிக்கையாக மாற்றும்போது ஏற்படுகிறது
415 Unsupported Media Type ஊடக வகை ஆதரிக்கப்படாததால் சேவையகம் கோரிக்கையை ஏற்காது
416 Range Not Satisfiable வாடிக்கையாளர் கோப்பின் ஒரு பகுதியைக் கோரியுள்ளார், ஆனால் சேவையகம் அந்தப் பகுதியை வழங்க முடியாது
417 Expectation Failed சேவையகம் Expect request-header புலத்தின் தேவைகளைப் பூர்த்தி செய்ய முடியாது

5xx: சேவையக பிழை

செய்தி விளக்கம்
500 Internal Server Error ஒரு பொதுவான பிழை செய்தி, இன்னும் குறிப்பிட்ட செய்தி பொருத்தமாக இல்லாதபோது வழங்கப்படுகிறது
501 Not Implemented சேவையகம் கோரிக்கை முறையை அங்கீகரிக்கவில்லை அல்லது கோரிக்கையை நிறைவேற்றும் திறன் இல்லை
502 Bad Gateway சேவையகம் ஒரு கேட்வே அல்லது ப்ராக்ஸியாக செயல்பட்டு, மேல்நிலை சேவையகத்திலிருந்து தவறான பதிலைப் பெற்றது
503 Service Unavailable சேவையகம் தற்போது கிடைக்கவில்லை (சுமை அதிகமாக உள்ளது அல்லது இயங்கவில்லை)
504 Gateway Timeout சேவையகம் ஒரு கேட்வே அல்லது ப்ராக்ஸியாக செயல்பட்டு, மேல்நிலை சேவையகத்திலிருந்து சரியான நேரத்தில் பதிலைப் பெறவில்லை
505 HTTP Version Not Supported சேவையகம் கோரிக்கையில் பயன்படுத்தப்படும் HTTP நெறிமுறை பதிப்பை ஆதரிக்காது
511 Network Authentication Required வாடிக்கையாளர் பிணைய அணுகலைப் பெற அங்கீகாரம் செய்ய வேண்டும்
