HTML Global Attributes

பொது பண்புகள் என்பது அனைத்து HTML கூறுகளுடனும் பயன்படுத்தக்கூடிய பண்புகளாகும்.

HTML Global Attributes Table

பண்பு விளக்கம்
accesskey ஒரு கூறை செயல்படுத்த/கவனம் பெற ஒரு குறுக்குவழி விசையைக் குறிப்பிடுகிறது
class ஒரு கூறுக்கு ஒன்று அல்லது அதற்கு மேற்பட்ட வகைப் பெயர்களைக் குறிப்பிடுகிறது (பாணி தாளில் உள்ள வகையைக் குறிக்கிறது)
contenteditable ஒரு கூறின் सामग्री திருத்தக்கூடியதா இல்லையா என்பதைக் குறிப்பிடுகிறது
data-* பக்கத்திற்கு அல்லது பயன்பாட்டிற்கு தனிப்பட்ட தனிப்பயன் தரவைச் சேமிக்கப் பயன்படுகிறது
dir ஒரு கூறில் உள்ள உள்ளடக்கத்திற்கான உரை திசையைக் குறிப்பிடுகிறது
draggable ஒரு கூறு இழுக்கக்கூடியதா இல்லையா என்பதைக் குறிப்பிடுகிறது
enterkeyhint மெய்நிகர் விசைப்பலகையில் உள்ள எண்டர்-கீயின் உரையைக் குறிப்பிடுகிறது
hidden ஒரு கூறு இன்னும் பொருத்தமானதல்ல அல்லது இனி பொருத்தமானதல்ல என்பதைக் குறிப்பிடுகிறது
id ஒரு கூறுக்கான தனிப்பட்ட அடையாளத்தைக் குறிப்பிடுகிறது
inert உலாவி இந்தப் பிரிவை புறக்கணிக்க வேண்டும் என்பதைக் குறிப்பிடுகிறது
inputmode மெய்நிகர் விசைப்பலகையின் பயன்முறையைக் குறிப்பிடுகிறது
lang கூறின் உள்ளடக்கத்தின் மொழியைக் குறிப்பிடுகிறது
popover பாப்ஓவர் கூறைக் குறிப்பிடுகிறது
spellcheck கூறின் எழுத்துப்பிழை மற்றும் இலக்கணம் சரிபார்க்கப்பட வேண்டுமா இல்லையா என்பதைக் குறிப்பிடுகிறது
style ஒரு கூறுக்கான இன்லைன் CSS பாணியைக் குறிப்பிடுகிறது
tabindex ஒரு கூறின் டேபிங் வரிசையைக் குறிப்பிடுகிறது
title ஒரு கூறு பற்றிய கூடுதல் தகவலைக் குறிப்பிடுகிறது
translate ஒரு கூறின் सामग्री மொழிபெயர்க்கப்பட வேண்டுமா இல்லையா என்பதைக் குறிப்பிடுகிறது

Exercises

❓ Which of the following is a global attribute?

class
✓ சரி! class is a global attribute
href
✗ தவறு! href is not a global attribute, it's specific to <a> and <link> elements
id
✓ சரி! id is a global attribute
src
✗ தவறு! src is not a global attribute, it's specific to media elements
