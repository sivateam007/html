HTML तत्व में पृष्ठभूमि छवि

किसी HTML तत्व में पृष्ठभूमि छवि जोड़ने के लिए, HTML शैली विशेषता और CSS पृष्ठभूमि-छवि विशेषता का उपयोग करें:

उदाहरण

<p> तत्व में एक पृष्ठभूमि छवि जोड़ें:

<p style="background-image: url('img_girl.jpg');">

आप <head> अनुभाग में <style> तत्व में एक पृष्ठभूमि छवि भी निर्दिष्ट कर सकते हैं:

उदाहरण

<style>
p {
  background-image: url('img_girl.jpg');
}
</style>

पृष्ठ पर पृष्ठभूमि छवि

यदि आप पूरे पृष्ठ के लिए पृष्ठभूमि छवि चाहते हैं, तो <body> तत्व में पृष्ठभूमि छवि निर्दिष्ट करें:

उदाहरण

<style>
body {
  background-image: url('img_girl.jpg');
}
</style>

पृष्ठभूमि छवि को दोहराते हुए

यदि पृष्ठभूमि छवि तत्व से छोटी है, तो छवि स्वचालित रूप से क्षैतिज और लंबवत रूप से दोहराई जाएगी जब तक कि यह तत्व के अंत तक नहीं पहुंच जाती:

उदाहरण

<style>
body {
  background-image: url('example_img_girl.jpg');
}
</style>

बैकग्राउंड छवि को दोहराने से रोकने के लिए, बैकग्राउंड-रिपीट विशेषता को नो-रिपीट पर सेट करें:

उदाहरण

<style>
body {
  background-image: url('example_img_girl.jpg');
  background-repeat: no-repeat;
}
</style>

पृष्ठभूमि छवि पर फैलाएं

यदि आप चाहते हैं कि पृष्ठभूमि छवि पूरे तत्व को कवर करे, तो आप पृष्ठभूमि-आकार गुण को कवर करने के लिए सेट कर सकते हैं।

साथ ही, यह सुनिश्चित करने के लिए कि पूरा तत्व हमेशा कवर किया गया है, बैकग्राउंड-अटैचमेंट विशेषता को फिक्स्ड पर सेट करें:

उदाहरण

<style>
body {
  background-image: url('img_girl.jpg');
  background-repeat: no-repeat;
  background-attachment: fixed;
  background-size: cover;
}
</style>

पृष्ठभूमि छवि का विस्तार

यदि आप चाहते हैं कि पृष्ठभूमि छवि पूरे तत्व में फिट होने के लिए खिंच जाए, तो आप पृष्ठभूमि-आकार की संपत्ति को 100% 100% पर सेट कर सकते हैं:

उदाहरण

<style>
body {
  background-image: url('img_girl.jpg');
  background-repeat: no-repeat;
  background-attachment: fixed;
  background-size: 100% 100%;
}
</style>
